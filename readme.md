## Código Criado na Disciplina de Pensamento Computacional"

Arthur Tramontina Saggin - 1139361

Jogo baseado em IronManV2, com modificações em cenários, personagens, com o mesmo desafio de esquivas mas com um grau mais!

Foi utilizado as seguintes funções:

* Pygame
* SQLite (armazenamento de recordes e pontuações)
* Programação Orientada a Eventos
* Detecção de Colisões (pygame.Rect e colliderect)
* Geração de Números Aleatórios (módulo random)
* Manipulação de Imagens e Sprites
* Áudio Digital (efeitos sonoros e música de fundo)
* Interface Gráfica para Jogos 2D
* Animações e Movimentação de Objetos em Tempo Real

Contribuição realizada por João Isaque.
